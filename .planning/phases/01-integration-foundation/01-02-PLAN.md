  <how-to-verify>
**Step 1: Add MQTT Integration**
1. Open https://hass.shad-bangus.ts.net
2. Go to Settings > Devices and Services > Add Integration
3. Search for "MQTT" and select it
4. Enter broker: `127.0.0.1`, port: `1883`
5. Leave username and password empty (local-only broker, no auth)
6. Click Submit

**Step 2: Add Frigate Integration**
1. Still in Settings > Devices and Services > Add Integration
2. Search for "Frigate" and select it
3. Enter URL: `http://127.0.0.1:5000` (direct API port, NOT port 80)
4. Click Submit

**Step 3: Verify Entity Discovery**
1. Go to Settings > Devices and Services > Frigate
2. Confirm it shows devices for driveway, front_door, garage
3. Click into one device and verify entities exist:
   - `camera.driveway` (camera entity)
   - `binary_sensor.driveway_motion` (motion sensor)
   - `switch.driveway_detect` (detection toggle)
   - `sensor.driveway_person_count` (or similar detection sensors)
4. Go to Developer Tools > States
5. Filter by "frigate" -- confirm entities have real states (not "unavailable")
6. Wave your hand in front of a camera and confirm the motion binary sensor changes state within a few seconds

Type "approved" if entities are discovered and updating in real-time. If issues arise, describe what you see.
