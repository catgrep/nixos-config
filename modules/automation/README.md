# Home-Assistant: Integrating Third-Party Smart Home Devices

| Ecosystem | LAN/Local Integration | Protocol |
|---|---|---|
| **Govee Home** | `govee-lan` (HACS) or `govee2mqtt` (HACS) | LAN UDP / MQTT |
| **TP-Link Tapo** | Built-in `tplink` integration or `tapo` (HACS) | LAN (proprietary) |
| **Yale/August Locks** | HomeKit Controller (built-in) or Z-Wave (built-in) | HomeKit (BLE/Wi-Fi) / Z-Wave |
| **Lutron Caséta** | Built-in `lutron_caseta` integration | Lutron LEAP (LAN) |
